# AT72 Verse — Vision

AT72 Verse est une plateforme SaaS d'agents IA destinée aux entreprises.

Elle ne se limite pas à un chatbot : elle offre un écosystème d'agents spécialisés
capables de collaborer, orchestrés par Adam, sur les fondations **Verse Core** et **Verse Kernel**.

## Principes

- Core ≠ Agent
- Kernel-mediated I/O
- Persona & Skills configurables
- Marketplace-ready dès les fondations
